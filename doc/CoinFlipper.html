    Method that calls a random (between 0 and 1) in order to determine the result of the coin flip 1 equals to head and 0 to tails.

    # File models/clases.rb, line 224
def flip_coin (room , game)
  if (rand(0...2) == 1)
    @head.call(room, game)
  else
    @tail.call(room, game)
  end
end
